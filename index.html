Refresh settings.json - High Performance Banners

Instructions:

1. Copy the settings.json file from the old package and paste its content into the left box. 2. Copy the settings.json file from the new package and paste its content into the right box. 3. Press the button "Generate Updated Version". 4. Review the Results:
- On the left side, you’ll see the original settings from the new package you created.
- On the right side, you’ll see the updated settings with highlighted values merged from the old package. 5. Copy the content from the right box ("Updated Settings") and paste it into the new package’s JSON file to apply the updates.
Latest HP version supported/tested: *_2024-10-04
